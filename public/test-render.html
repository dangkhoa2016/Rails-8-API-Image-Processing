Image Render Smoke Test

Trang này dùng Vue 3 CDN, không cần build tool. Luồng test: đăng nhập lấy JWT, nhập image URL, sau đó tải từng biến thể render ảnh theo thứ tự để xem tốc độ, lỗi xử lý và kết quả preview từ API.

Khuyến nghị khi test

Sau khi tạo danh sách test, hãy bấm nút chạy theo đúng nhóm scenario bạn cần kiểm tra. Cách này giúp tránh bắn toàn bộ request một lúc, giảm nguy cơ rate limit và tiết kiệm tài nguyên khi bạn không cần test hết mọi nhóm.

Đăng nhập

2

Nhập image URL

Sau khi đăng nhập thành công mới cho tạo danh sách render.

Mỗi card bên dưới hiển thị đầy đủ GET URL của API. Ảnh được tải bằng fetch kèm Bearer token, vì <img src> không tự gắn header Authorization.

Mặc định trên domain public sẽ dùng same-origin static image để tránh hotlink rate-limit. Nếu bạn nhập URL từ các host như Wikimedia, một số host có thể trả về 429 sau vài request liên tiếp. Trên localhost, không dùng được ảnh cùng origin vì SSRF guard chặn loopback.
